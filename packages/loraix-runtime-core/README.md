# Loraix Runtime Core (`loraix-runtime-core`)

Provider-agnostic core runtime package for Loraix SDK.

## Install

```bash
npm install loraix-runtime-core
```

## Usage

```js
import { LoraixRuntime } from 'loraix-runtime-core';
```
