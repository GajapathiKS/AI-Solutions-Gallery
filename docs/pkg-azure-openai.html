Package

loraix-provider-azure-openai

Azure OpenAI deployment provider

Install

npm install loraix-provider-azure-openai

Usage

import { AzureOpenAIProvider } from 'loraix-provider-azure-openai';
const provider = new AzureOpenAIProvider({ apiKey, resourceName, deployment });

Highlights

  • deployment endpoint routing
  • chat + stream support
  • Azure-specific request shape

Integrate with Runtime

import { LoraixRuntime } from 'loraix-runtime-core';
const ai = new LoraixRuntime({ provider, model: 'gpt-4o-mini' });
const out = await ai.generate('Hello');
