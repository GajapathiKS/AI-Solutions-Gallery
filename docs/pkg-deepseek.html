Package

loraix-provider-deepseek

DeepSeek provider adapter

Install

npm install loraix-provider-deepseek

Usage

import { DeepSeekProvider } from 'loraix-provider-deepseek';
const provider = new DeepSeekProvider({ apiKey: process.env.DEEPSEEK_API_KEY });

Highlights

  • OpenAI-compatible endpoint use
  • chat + stream support
  • portable runtime integration

Integrate with Runtime

import { LoraixRuntime } from 'loraix-runtime-core';
const ai = new LoraixRuntime({ provider, model: 'gpt-4o-mini' });
const out = await ai.generate('Hello');
