Package

loraix-provider-openai

OpenAI provider adapter

Install

npm install loraix-provider-openai

Usage

import { OpenAIProvider } from 'loraix-provider-openai';
const provider = new OpenAIProvider({ apiKey: process.env.OPENAI_API_KEY });

Highlights

  • chat completions mapping
  • SSE chunk streaming
  • status-aware provider errors

Integrate with Runtime

import { LoraixRuntime } from 'loraix-runtime-core';
const ai = new LoraixRuntime({ provider, model: 'gpt-4o-mini' });
const out = await ai.generate('Hello');
