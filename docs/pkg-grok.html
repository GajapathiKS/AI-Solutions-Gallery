Package

loraix-provider-grok

xAI Grok provider adapter

Install

npm install loraix-provider-grok

Usage

import { GrokProvider } from 'loraix-provider-grok';
const provider = new GrokProvider({ apiKey: process.env.XAI_API_KEY });

Highlights

  • OpenAI-compatible transport
  • SSE stream parsing
  • provider error mapping

Integrate with Runtime

import { LoraixRuntime } from 'loraix-runtime-core';
const ai = new LoraixRuntime({ provider, model: 'gpt-4o-mini' });
const out = await ai.generate('Hello');
