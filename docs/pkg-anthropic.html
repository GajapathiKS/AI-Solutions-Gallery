Package

loraix-provider-anthropic

Anthropic Claude provider adapter

Install

npm install loraix-provider-anthropic

Usage

import { AnthropicProvider } from 'loraix-provider-anthropic';
const provider = new AnthropicProvider({ apiKey: process.env.ANTHROPIC_API_KEY });

Highlights

  • messages API mapping
  • event + data SSE parsing
  • usage normalization

Integrate with Runtime

import { LoraixRuntime } from 'loraix-runtime-core';
const ai = new LoraixRuntime({ provider, model: 'gpt-4o-mini' });
const out = await ai.generate('Hello');
