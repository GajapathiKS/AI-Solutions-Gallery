Package

loraix-provider-bedrock

AWS Bedrock SDK provider

Install

npm install loraix-provider-bedrock

Usage

import { BedrockProvider } from 'loraix-provider-bedrock';
const provider = new BedrockProvider({ region: 'us-east-1' });

Highlights

  • Converse/ConverseStream APIs
  • SigV4 auth via SDK credentials
  • HTTP status propagation for retries

Integrate with Runtime

import { LoraixRuntime } from 'loraix-runtime-core';
const ai = new LoraixRuntime({ provider, model: 'gpt-4o-mini' });
const out = await ai.generate('Hello');
