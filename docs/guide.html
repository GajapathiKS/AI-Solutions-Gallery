Getting Started

1) Install runtime + provider

npm install loraix-runtime-core loraix-provider-openai

2) Create runtime client

import { LoraixRuntime } from 'loraix-runtime-core';
import { OpenAIProvider } from 'loraix-provider-openai';

const ai = new LoraixRuntime({
  provider: new OpenAIProvider({ apiKey: process.env.OPENAI_API_KEY }),
  model: 'gpt-4o-mini',
  maxRetries: 2,
  timeoutMs: 20000
});

3) Generate, JSON mode, and stream

const text = await ai.generate('Explain recursion in 2 lines');
const profile = await ai.json({ prompt: 'user profile', schema: { name: 'string', age: 'number' } });
for await (const chunk of ai.stream('Write a haiku')) process.stdout.write(chunk);

4) Choose another provider

Swap provider package only; runtime code remains stable. See package docs.
