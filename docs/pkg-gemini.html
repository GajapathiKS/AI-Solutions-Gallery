Package

loraix-provider-gemini

Google Gemini provider adapter

Install

npm install loraix-provider-gemini

Usage

import { GeminiProvider } from 'loraix-provider-gemini';
const provider = new GeminiProvider({ apiKey: process.env.GEMINI_API_KEY });

Highlights

  • generateContent mapping
  • stream chunk handling
  • usage output shaping

Integrate with Runtime

import { LoraixRuntime } from 'loraix-runtime-core';
const ai = new LoraixRuntime({ provider, model: 'gpt-4o-mini' });
const out = await ai.generate('Hello');
