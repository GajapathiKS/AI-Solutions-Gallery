Runtime API Reference

Core Methods

MethodDescription
generate(input, options?)Generate text from prompt or messages.
json({ prompt, schema, ... })Strict JSON generation with shallow schema validation and one repair retry.
stream(input, options?)Async iterable chunks; timeout/retry protection around setup and stream handling.

Config

{
  provider, model,
  temperature?, maxTokens?, timeoutMs?,
  maxRetries?, retryStrategy?, fallbackProviders?
}

Retry Rules

  • Retryable: 429, 500, 503, timeout, network failures.
  • Non-retryable: 400/auth/json-schema validation failures.
  • Fallback providers are used only after retry exhaustion.

Provider Contract

provider.generate(request, ctx) => { text, usage?, raw }
provider.stream(request, ctx) => AsyncIterable<string>
