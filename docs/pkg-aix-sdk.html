Package

aix-sdk

Facade over runtime + providers

Install

npm install aix-sdk

Usage

import { createAixClient } from 'aix-sdk';
const ai = await createAixClient({ provider: 'openai', apiKey: process.env.OPENAI_API_KEY, model: 'gpt-4o-mini' });

Highlights

  • quick bootstrap for app teams
  • provider factory helpers
  • typed wrappers for runtime creation

Integrate with Runtime

import { LoraixRuntime } from 'loraix-runtime-core';
const ai = new LoraixRuntime({ provider, model: 'gpt-4o-mini' });
const out = await ai.generate('Hello');
